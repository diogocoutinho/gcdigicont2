Como criar o arquivo og-image.png

Opção 1: Screenshot Manual

  1. Abra este arquivo HTML em seu navegador
  2. Use uma ferramenta de screenshot para capturar apenas a área azul (1200x630px)
  3. Salve como og-image.png na pasta public/

Opção 2: Usar uma ferramenta online

  1. Acesse Canva.com ou Figma.com
  2. Crie um design de 1200x630 pixels
  3. Use as cores: Background #0e2b3a, Accent #c79d4b
  4. Adicione o logo e texto conforme o template acima
  5. Exporte como PNG e salve em public/og-image.png

Opção 3: Browser DevTools

  1. Abra DevTools (F12)
  2. Clique com o botão direito no elemento com id="og-image"
  3. Selecione "Capture node screenshot" (Chrome) ou similar
  4. Salve como og-image.png na pasta public/

Depois de criar a imagem, você pode deletar este arquivo HTML.
